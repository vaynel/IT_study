# 가상화(Virtualization) 공부 플랜

## 목표
- 가상화의 기본 개념과 종류를 이해한다. (하이퍼바이저, 컨테이너, 하드웨어 가상화 등)
- 주요 하이퍼바이저(KVM, Xen, VMware, Hyper-V)와 오픈소스 툴(libvirt, QEMU)의 사용법을 실습으로 익힌다.
- 네트워크/스토리지/성능/보안 관점에서 가상화 운영 능력을 갖춘다.

---

## 추천 학습 기간: 4주 (주당 8~12시간 권장)

### 주차별 계획

- **1주차 — 기본 개념 & 환경 준비**
  - 가상화 정의, 목적, 장단점 학습
  - 하이퍼바이저의 종류(타입1/타입2)와 구조 이해
  - 컨테이너 vs 가상머신 비교
  - 실습: Ubuntu(또는 CentOS)에 KVM/Libvirt/QEMU 설치 및 `virt-manager`로 간단한 VM 생성

- **2주차 — 하이퍼바이저 & VM 관리 심화**
  - `qemu-img`, `virt-install`, `virsh` 명령어 실습
  - 이미지 관리(회복, 스냅샷) 및 클론 전략
  - 스토리지 백엔드 (qcow2, raw, LVM, Ceph RBD) 개요
  - 실습 과제: 이미지 변환, 스냅샷 생성/복원, LVM 기반 VM 생성

- **3주차 — 네트워크 가상화 & 성능**
  - 브리지, macvlan/macvtap, SR-IOV 개념과 사용 시나리오
  - 가상 스위치(OVS) 기본 설정과 트래픽 제어
  - 성능 측정 도구(fio, iperf, perf)로 I/O와 네트워크 성능 테스트
  - 실습 과제: 브리지 구성, OVS로 가상 네트워크 구성, 성능 비교 리포트 작성

- **4주차 — 보안, 고급 주제, 클라우드 연계**
  - 게스트 보안(SELinux/AppArmor, VM 격리), 이미지 서명, 네트워크 정책
  - 라이브 마이그레이션, HA(High Availability), 리소스 스케줄링
  - 클라우드에서의 가상화 이해 (AWS EC2, Hypervisor 비교, 가상화 기술 선택 이유)
  - 실습 과제: 간단한 라이브 마이그레이션 테스트 또는 가상화 기반 장애/복구 시나리오 수행

---

## 실습 과제 (권장)
- 로컬에서 KVM 기반 VM 3개를 만들고 서로 다른 네트워크(브리지/네트워크 분리)에 연결해 통신 테스트
- VM 이미지 최적화: 불필요한 패키지 제거, 클라우드 이미지 커스터마이즈
- 컨테이너와 VM의 부팅/리소스 사용 비교(시작 시간, 메모리, CPU 사용량)
- 간단한 자동화: `virt-install` 스크립트로 인프라 코드 형태로 VM 배포

---

## 평가/완성 기준
- VM을 생성/삭제/스냅샷/복원할 수 있다
- 네트워크 가상화(브리지, OVS 등) 기본 구성과 트러블슈팅을 수행할 수 있다
- 성능 테스트 결과를 해석해 병목 원인을 제시할 수 있다
- 클라우드에서의 가상화 특성을 설명하고, 운영 시 고려사항을 말할 수 있다

---

## 참고 자료 (시작용)
- KVM/QEMU: 공식 문서 및 `libvirt` 매뉴얼
- "Linux KVM" 관련 튜토리얼(공식 위키/배포판 문서)
- Open vSwitch (OVS) 가이드
- 각 하이퍼바이저 벤더 문서(ESXi, Hyper-V)
